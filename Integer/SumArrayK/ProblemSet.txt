Subarray Sum K
Question
GeeksforGeeks: Find subarray with given sum - GeeksforGeeks
Problem Statement
Given an nonnegative integer array, find a subarray where the sum of numbers is k. Your code should
return the index of the first number and the index of the last number.
Example
Given [1, 4, 20, 3, 10, 5] , sum k = 33, return [2, 4]
